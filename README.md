# simple-python-app-codespaces-v3

This repo is based on the simple-python-app-codespaces-v2 template. simple-python-app-codespaces-v3 adds a github action that builds, cosigns and push the built image to github packages.
If building a PR signing and pushing is skipped.

Furthermore a badge indicating if build and push was successful is added.
